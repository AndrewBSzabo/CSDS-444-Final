# CSDS-444-Final

Vigenere Cipher
DES
RSA

To run application:
1. cd into CSDS-444-Final/

2. Create a virtualenv (and activate it):

python3 -m venv venv

. venv/bin/activate

3. Install the flask app:

pip install -e .

4. Run the server:

export FLASK_APP=comp_sec

export FLASK_ENV=development

flask run

5. Go to http://127.0.0.1:5000/ in browser
